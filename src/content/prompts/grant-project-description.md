## The Prompt

```
Write a project description for [PROJECT/PROGRAM NAME] for a grant to [FUNDER NAME].

The description should:
- Describe program goals and objectives
- Explain methodology and activities: [KEY ACTIVITIES]
- Identify target population and number served: [POPULATION]
- Provide timeline: [TIMEFRAME]
- Describe staff and organizational capacity
- Explain how success will be measured: [EVALUATION METHODS]
- Connect to funder priorities
- Be 500-800 words
- Use [PROFESSIONAL/DETAILED/CLEAR] tone
```
