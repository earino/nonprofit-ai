## The Prompt

```
Write a needs statement for a grant proposal to [FUNDER NAME] addressing [PROBLEM/ISSUE].

The statement should:
- Lead with compelling data or story illustrating the problem
- Quantify the need with local statistics: [LOCAL DATA/STATISTICS]
- Demonstrate understanding of root causes
- Show gap between current state and desired state
- Connect to funder's priorities: [FUNDER PRIORITIES]
- Establish urgency without being melodramatic
- Cite credible sources
- Be 300-500 words
- Use [COMPELLING/DATA-DRIVEN/PROFESSIONAL] tone
```
