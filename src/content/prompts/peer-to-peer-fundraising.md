## The Prompt

```
Create a template for supporters to use when asking their network to support [ORGANIZATION NAME].

The template should:
- Explain why [FUNDRAISER NAME] personally supports this cause
- Share connection to the mission: [PERSONAL CONNECTION]
- Set a fundraising goal: [GOAL AMOUNT]
- Explain what funds will support: [PROGRAM/PURPOSE]
- Make it easy for friends/family to donate
- Be 200-300 words
- Use a [PERSONAL/AUTHENTIC/FRIENDLY] tone
- Be customizable for different supporters
```

## How to Customize

1. Replace all [BRACKETED] fields with your specific information
2. Adjust tone and length as needed for your audience
3. Review and personalize before using
